Flask基础2

1.装饰器

  1. 装饰器的作用:在不改变原有功能代码的基础上,添加额外的功能,如用户验证等
  2. 构成装饰器的条件
  • 外层函数前套内层函数
  • 外层函数返回内层函数
  • 内层函数调用外层函数的参数
    注意：
  • 外层函数传入的参数即为被装饰的函数
  • 添加装饰器后外层装饰的函数意义改变需用wraps不改变使用装饰器原有函数的结构(如name, doc)
    from  functools import wraps
@wraps(func)

    2.蓝图

  1. 产生背景: 管理路由地址
  2. 安装: pip install flask-blueprint
  3. 使用方法：
    第一步: blue = Blueprint('first', __name__) @blue.route('路由地址')
    第二步: flask对象app.register_blueprint(blueprint=blue)
  4. 页面跳转
    有参跳转：redirect(url_for('生成蓝图的第一个参数.跳转的函数名称', 参数名=值, 参数名2=值2))
    无参跳转：redirect(url_for('生成蓝图的第一个参数.跳转的函数名称'))

    3.模板

    定义网站模板的大体框架，定义可以被动态填充内容的block块
    block块定义: 
    {% block name %}
{% endblock %}
    注意:block名称不能取同名

    {% 标签名 %}
