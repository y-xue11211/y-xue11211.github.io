1.flask介绍

Flask是一个基于Python实现的web开发的’微’框架

flask流行的主要原因：

a）有非常齐全的官方文档，上手非常方便

b) 有非常好的拓展机制和第三方的拓展环境，工作中常见的软件都有对应的拓展，自己动手实现拓展也很容易

c) 微型框架的形式给了开发者更大的选择空间

2.环境安装

  1. 使用virtualenv创建

    安装: pip install virtualenv

    创建虚拟环境: virtualenv --no-site-packages -p D:\python37\python.exe flaskenv9

    激活环境:

    win: cd flaskenv9/Scripts 执行activate
    Linux/Mac:source activate

    退出环境: deactivate
  2. 使用pycharm创建

    创建新工程选择虚拟环境；

    3.pip的使用

    查询安装的库: pip list / pip freeze

    安装内容: pip install xxx

    安装文件: pip install -r requeirement.txt
    txt文档里可放多个需要安装的文件名

    4.flask安装与使用

    安装: 先激活环境，在执行pip install flask

    5.启动flask

  3. flask自带启动方式: app.run(host, port, debug)
  4. 使用flask-script库

    安装flask-scriptpip install flask-script
    from flask_script import Manager manege = Manager(app) manage.run()
    启动命令: python xxx.py runserver -p -h -d

    6.路由规则

    定义格式: <转换器(类型):参数名>

    转换器:int: 指定整型

       string: 指定字符串，默认类型

       float: 指定浮点类型
    注意: <string:name> 其实和<name>是一样的

    7.请求

  5. 获取参数

    GET请求:获取参数: 
    request.args 
request.args.get('key')
    POST请求获取参数: 
    request.form
request.form.get(key) 
#或者request.form[key]</br>
    获取多个同样的key对应的vallue值时，使用getlist(key)
  6. 获取请求路径: request.path
  7. 请求cookies: request.cookies
  8. 获取上传文件: request.files

    取值: request.files.get(key)或者 request.files[key]

    8.响应

    导包: from flask import make_response
    使用: make_response(‘响应内容’, 响应状态码)

    9.cookie

    产生场景: 由于HTTP无状态协议, cookie的使用，保存一个令牌（标识符）用于标识用户的登录状态

    令牌的产生: 在登录时向cookie中设置的。

    设置令牌: res = make_response() res.set_cookie(key,value, max_age)

    删除: 
    res=make_response()   
res.delete_cookie(key)
    注意:1.cookie不能跨浏览器
      2.cookie中设置的参数有字节限制，不能无限的存储参数

    10.session

    flask默认存储session的形式，将session数据保存在客户端:
      导包: from flask import session
      使用: session[key]=value
    使用flask-session库，将session数据保存在服务器端
    pip install redis flask-session
from flask_session import Session 
app.secret_key = '1234567890'  # 加密复杂程度
app.config['SESSION_TYPE'] ='redis'
app.config['SESSION_REDIS'] = redis.Redis()
Session(app)
